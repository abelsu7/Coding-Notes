# Git 常用命令

## git remote 远程仓库

```shell
# 查看关联的远程仓库名称
git remote
# 查看关联的远程仓库详细信息
git remote -v
# 添加远程仓库的关联
git remote add <name> <url>
# 删除远程仓库的关联
git remote remove <name>
# 修改远程仓库的关联
git remote set-url <name> <url>
```
