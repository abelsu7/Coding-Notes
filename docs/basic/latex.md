## 分数

| 格式 | 语法 | 示例 | 说明 |
| :-- | :-- | :-: | :-- |
| 分数 | `\frac{}{}` | $$\frac{1}{x^2-4}$$ | 行内分数会缩小以适应行高 |
| 调整分数大小 | `\displaystyle{}` | $$\displaystyle{\frac{1}{x^2-4}}$$ | 适用于行内分数 |
