| Keys | Function |
| :-- | :-- |
| Double **Shift** | Search Everywhere |
| **Alt+**数字 | 切换面版 |
| **Alt+Shift** | 移动当前行 |
| **Ctrl+Alt+L** | 格式化代码 |
| **Ctrl+Shift+A** | Find Action |
| **Ctrl+D** | 复制当前行 |
| **Ctrl+E** | 查看最近的文件 |
| **Ctrl+N** | Find a class, file, or symbol |
| **Ctrl+Y** | 删除选中行 |
| **Ctrl+(Shift)+W** | Extend or shrink selection |
| **Ctrl+Shift+F7** | Highlight usages in a file |
