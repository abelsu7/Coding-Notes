## VS Code 快捷键

<kbd>Ctrl</kbd>+ <kbd>P</kbd>：转到文件

<kbd>Ctrl</kbd>+ <kbd>Shift</kbd>+ <kbd>方向键</kbd>：选择多行

<kbd>F3</kbd>：查找下一个匹配

<kbd>Ctrl</kbd>+ <kbd>Shift</kbd>+ <kbd>K</kbd>：删除当前行
